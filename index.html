Questão 1

Crie um programa que gerencie o transporte de viaturas de sua Unidade. Seu programa deverá verificar se a capacidade de transporte de uma embarcação comporta o peso total das viaturas que serão transportadas. Para conseguir realizar o transporte, o peso total das viaturas transportadas deve ser igual ou menor que a capacidade de transporte da embarcação.

Os dados das viaturas estão cadastrados e armazenados em um dicionário, onde a chave é o nome da viatura e o valor é o seu respectivo peso, em toneladas. Observe um extrato desses dados:

viaturas = {"Guarani": 18.1, "M113": 12.3, "Guaicurus": 6.5, "Astros": 22.0, "M60": 56.2, "M577 A2": 7.2, "Cascavel": 12.1, "M578": 40.0, "Leopard": 42.3, "Centauro": 24.0, "Urutu": 13.0, "VBE Socorro": 40.2}

Seu programa deverá permitir que o usuário informe o nome da viatura que se deseja transportar e a quantidade. É possível escolher mais de um tipo de viatura a ser transportada. Para encerrar o programa, o usuário deve digitar sair.

Por fim, o usuário deverá informar a capacidade de transporte da embarcação, em toneladas, e seu programa deverá informar se é possível ou não realizar o transporte.

Nas impressões, formate os pontos flutuantes com 2 casas decimais.

Ver Gabarito 
viaturas = {"Guarani": 18.1, "M113": 12.3, "Guaicurus": 6.5, "Cascavel": 12.1, "M578": 40.0, "Leopard": 42.3}

total = 0
viatura = input('Nome da viatura ou sair: ')

while viatura != 'sair':
    qtd = int(input('Quantidade: '))
    total += viaturas[viatura] * qtd
    viatura = input('Nome da viatura ou sair: ')
    
maximo = float(input('Digite a capacidade de transporte da embarcação (ton): '))

print(f'O peso total das viaturas é igual a {total:.2f} toneladas.')
if total <= maximo:
    print('É possível transportar.')
else:
    print('Não é possível transportar.')

Questão 4

Uma disciplina da AMAN possui 2 avaliações de acompanhamento (AA) e uma avaliação de controle (AC). Caso o Cadete obtenha média maior ou igual a sete, ele passa direto. Caso contrário, realizará uma avaliação de controle final (ACF).

A média é calculada da seguinte forma: a média das duas AA possui peso 1 e a nota da AC possui peso 2. Crie um programa que calcule a média dessa disciplina e informe se o Cadete passou direto ou realizará ACF.

O formato de entrada será uma única string contendo o nome do cadete, a nota da AA1, a nota da AA2 e a nota da AC, todos separados por hífen. Poderão ser informados os dados de mais de um Cadete na mesma linha.
Exemplo de entrada: João-7.2-8.3-7.4-Maria-2.6-6.7-5.7

Seu programa deverá imprimir uma lista de tuplas, onde cada tupla conterá o nome do Cadete, a média obtida (arredondada para uma casa decimal) e a sua situação ("Passou direto." ou "Realizará ACF."). Todos os dados dentro da tupla devem ser do tipo string.

Exemplo de Entrada Exemplo de Saída
Paulo-3.5-6.7-2.1 Digite os dados: Paulo-3.5-6.7-2.1 [('Paulo', '3.1', 'Realizará ACF.')]
Andriole-4.5-9.5-7.0 Digite os dados: Andriole-4.5-9.5-7.0 [('Andriole', '7.0', 'Passou direto.')]
João-7.2-8.3-7.4-Maria-2.6-6.7-5.7 Digite os dados: João-7.2-8.3-7.4-Maria-2.6-6.7-5.7 [('João', '7.5', 'Passou direto.'), ('Maria', '5.4', 'Realizará ACF.')]
Pedro-5.7-9.7-5.2-Carlos-6.8-8.2-7.4-André-4.5-8.3-6.2-Ana-4.5-7.3-5.5 Digite os dados: Pedro-5.7-9.7-5.2-Carlos-6.8-8.2-7.4-André-4.5-8.3-6.2-Ana-4.5-7.3-5.5 [('Pedro', '6.0', 'Realizará ACF.'), ('Carlos', '7.4', 'Passou direto.'), ('André', '6.3', 'Realizará ACF.'), ('Ana', '5.6', 'Realizará ACF.')]
Ver Gabarito (Código Python) 
dados = input("Digite os dados: ")
dados = dados.split("-")
resultado_geral = []

for i in range(0, len(dados), 4):
    cadete = dados[i]
    AA1 = float(dados[i+1])
    AA2 = float(dados[i+2])
    media_AA = (AA1 + AA2) / 2
    AC = float(dados[i+3])
    
    media = (media_AA + 2 * AC) / 3
    
    if media >= 7:
        situacao = "Passou direto."
    else:
        situacao = "Realizará ACF."
        
    media = round(media, 1)
    media = str(media)
    resultado = (cadete, media, situacao)
    resultado_geral.append(resultado)

print(resultado_geral)

Exercício R.1: Filtrando Dicionários

Desenvolva uma função que receba um dicionário de alunos e suas respectivas notas, retornando apenas os nomes dos aprovados (nota maior ou igual a 7).

Entrada InternaSaída Esperada
{"Ana": 8.5, "Pedro": 6.0, "Maria": 7.5}['Ana', 'Maria']
Ver Gabarito 
def filtrar_aprovados(alunos):
    aprovados = []
    for nome, nota in alunos.items():
        if nota >= 7.0:
            aprovados.append(nome)
    return aprovados
